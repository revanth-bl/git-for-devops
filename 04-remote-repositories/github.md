# GitHub

GitHub is a cloud-based platform for hosting Git repositories and collaborating on software projects.

## Common GitHub Workflow

1. Create a repository.
2. Clone the repository.
3. Make changes.
4. Stage the changes.

```bash
git add .
```

5. Commit the changes.

```bash
git commit -m "Your commit message"
```

6. Push the changes.

```bash
git push origin main
```

GitHub also provides features like Pull Requests, Issues, Actions, and Discussions for team collaboration.
